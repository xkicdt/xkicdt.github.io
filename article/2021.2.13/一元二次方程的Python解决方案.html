一元二次方程的Python解决方案

前言

Python是一门高级脚本语言，简单易学，能完成许多普通任务。但这个高级语言并不是真的“功能多”，虽然它的库多，但是C++也有这么多库。写3D游戏必须用Java、C/C++等，脚本语言是不行的。还有操作系统，这个C++也少，多数为原汁原味的C。C用来写操作系统真的是得天独厚，Linux、Windows、XNU、Hurd、BSD、Unix全是C写的。但是C在完成简单任务时就会很复杂，这时Python的优势就显现出来。一元二次方程的解决不对时间、性能敏感，这种情况下使用Python非常合适。这篇文章我会把Python和C语言解法的代码都写出来，最后比较一下。

Python

import cmath;
a = int(input("Please input a = "))
b = int(input("Please input b = "))
c = int(input("Please input c = "))
delta = (b * b) - (4 * a * c)
if delta > 0:
 x1 = ((-b) + cmath.sqrt(delta)) / (2 * a)
 x2 = ((-b) - cmath.sqrt(delta)) / (2 * a)
 print("Two return: x1 = ", x1, ", x2 = "; x2)
if delta == 0:
 x1 = (-b) / (2 * a)
 print("One return: x1 = ", x1)
if delta < 0:
 print("No return")
exit(input(Press enter to exit))

C

#include "stdio.h"
#include "math.h"
int main(void)
{
 int a, b, c;
 printf("Please input a = ")
 scanf("%lf", &a);
 printf("Please input b = ");
 scanf("%lf", &b);
 printf("Please input c = ");
 scanf("%lf", &c);
 double x1, x2;
 double delta = (b * b) - (4 * a * c);
 if (delta > 0)
 {
  x1 = ((-b) + sqrt(delta)) / (2 * a);
  x2 = ((-b) - sqrt(delta)) / (2 * a);
  printf("Two return: x1 = %d; x2 = %d\n", x1, x2);
 }
 if (delta == 0)
 {
  x1 = (-b) / (2 * a);
  printf("One return: x1 = %d\n", x1);
 }
 if (delta < 0)
 {
  printf("No return\n");
 }
 return 0;
}

他们的原理相同，都是把一元二次方程转化成ax*x + bx + c = 0的格式，然后再用跟的判别式delta的(b * b) - (4 * a * c)形式，使用的是“公式法”求解。这是初中知识。显然，C的解法原理和Python一样，但是却多了一些代码。这种情况下Python更加合适。
